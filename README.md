# 鳄影斗兽场

一款基于 React 和 Vite 构建的迷你游戏。你被困在圆形斗兽场中，鳄鱼会从边缘不断生成并组成追击链跟踪你。使用弓箭击杀它们——每只鳄鱼需要命中 5 次才会死亡。避开鳄鱼，否则游戏结束。

## 操作说明

- **移动**：WASD 或方向键控制移动，按住鼠标/触屏也可朝目标方向移动
- **瞄准**：移动鼠标调整瞄准方向
- **射击**：点击或按空格键射箭

## 游戏规则

- 斗兽场边界会将你挡回，但不会造成伤害
- 鳄鱼数量会随时间不断增加
- 每只鳄鱼需要命中 5 次才能击杀
- 碰到鳄鱼的身体即失败

